Layer Operations

The above image can easily be created from three separate layers

The Layer Operations alter the currently selected layer (and in the case of the Flatten Operation, all the layers). This most not be confused with the Image Operations which alter all the layers at the same time.  This critical difference should be noted, for if this detail is ignored, using these options will yield strange results from what was expected.  All Layer Operations can be accessed through the Layers drop-down menu.
 
The following are the Image Operations:
  • Add New Layer
  • Delete Layer
  • Duplicate Layer
  • Flip Layer
  • Flatten Image
  • Layer Properties

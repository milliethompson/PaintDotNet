Layer Operations

The above image was created from three separate layers

The Layer Operations alter the currently selected layer (and in the case of the Flatten Operation, all the layers). This should not be confused with the Image Operations which alter all the layers at the same time.  This critical difference should be noted, for if this detail is ignored, using these options will yield strange results from what was expected.  All Layer Operations can be accessed through the Layers drop-down menu.

Layers with transparency may reveal the presence of a checkerboard pattern.  The checkerboard pattern represents the sub-background layer: a special layer that helps visualize areas of transparency that might otherwise be obscured by a white background.
 

The following are the Layer Operations:
  • Add New Layer
  • Delete Layer
  • Duplicate Layer
  • Import From File
  • Adjustments:
    • Auto-Level
    • Brightness/Contrast
    • Desaturate
    • Hue / Saturation
    • Invert Colors
    • Levels
    • Rotate / Zoom
  • Flatten Image
  • Flip Layer
  • Layer Properties

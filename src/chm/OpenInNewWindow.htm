Open in New Window

The Open in New Window Operation (File-->Open in New Window) prompts for an image to load and then starts a new instance of Paint.NET with that image.  The Open in New Window Operation is very similar to the New Window Operation, except instead of a blank canvas an image is loaded.
This is an example of performing an Open in New Window Operation while another image is currently being edited.  In this example, the user doesn't have to complete editing, and close the current image before they start working on another image.  Rather, separate images can be worked upon concurrently.

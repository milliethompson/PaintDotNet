Common Tools

The Common Tools are the primary ways in which you will be interacting with Paint.NET.  These tools can be selected from both the pull-down menu and the floating Tool Window.  You can also control the Aliasing attribute of each of the tools.  

For more information on a specific tool, please choose one of the following:

  • Rectangle Select Tool
  • Move Tool
  • Lasso Select Tool
  • Pencil Tool
  • Paint Brush Tool
  • Eraser Tool
  • Rectangle, Ellipse and Rounded Rectangle Tool
  • Line Tool
  • Free Form Shape Tool
  • Color Picker Tool
  • Paint Bucket Tool
  • Background/Foreground Color Selection Tool
  • Text Tool
  • Pan Tool

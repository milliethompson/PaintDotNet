Overview

Paint.NET is the new way to easily create and edit photos for a variety of purposes.  Maintaining some of the best features of the original MS-Paint application, Paint.NET also includes new, powerful tools.  Such new tools include an undo-redo history storage and a powerful layer tool which is often seen in high-quality photo editing applications.  This product was engineered using the .NET Framework, the C# programming language, and the GDI+ graphics libraries.

Overview

Welcome to the new way to easily create and edit your photos! Paint.NET builds upon the best features of the original Microsoft® Paint application and adds new tools with powerful features.  Two of the new tools include an interactive undo/redo history and a powerful layer engine only found in professional image editing applications.  Paint.NET was engineered using the Microsoft® .NET Framework, the C# programming language, and the GDI+ graphics libraries.

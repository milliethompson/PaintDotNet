Add New Layer Operation

The Add New Layer Operation (Layers-->Add New Layer) allows you to add another layer to the image.  This layer will start off transparent, visible, and selected in the Layers Window.  It is here that you may double click on the icon to set its Layer Properties.  Please follow the preceding link to learn more about that feature.
 
Note:  You can perform the Add New Layer Operation from either the drop-down menu (Layers-->Add New Layer) or from the Layers Window.

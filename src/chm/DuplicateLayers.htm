Duplicate Layer Operation

The Duplicate Layer Operation (Layers-->Duplicate Layer) performs an exact copy of the currently selected layer.  All properties of the layer (transparency and so forth) along with the graphical contents of the layer will be duplicated.  Don't be alarmed about the duplication of the name field of the layer.  This can easily be changed by double clicking on the new layer and changing that attribute.
 
Note:  You can perform the Duplicate Layer Operation from either the drop-down menu (Layers-->Duplicate Layer) or from the Layers Window.

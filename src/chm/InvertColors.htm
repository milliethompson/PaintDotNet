Invert Colors Effect

The Invert Colors Effect (Layers --> Adjustments --> Invert Colors) simply inverts the colors on the active layer.  This effect is very similar to a negative of a photo.  Applying operation a second time produces the original image again.

If there is a selection, this effect will only apply to that selection. Otherwise, this effect will apply to an entire active layer.

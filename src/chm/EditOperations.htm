Edit Operations

The Edit Operations allow for easy manipulation of the image history, the image itself, and the selected regions of the image.  Notice how the Edit drop-down menu is sectioned off into those topics. 
The following are the Edit Operations:
  • Undo
  • Redo
  • Cut
  • Copy
  • Paste
  • Paste in to New Layer
  • Erase Selection
  • Invert Selection
  • Select All
  • Deselect

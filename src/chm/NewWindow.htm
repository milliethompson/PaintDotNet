New Window

The New Window Operation (File-->New Window) opens up a new instance of Paint.NET.  This new instance of Paint.NET will have a clean canvas.  The New Window Operation is very similar to the Open in New Window Operation, except for the fact that in the latter, you are prompted for an image to load.
This is an example of doing a New Window Operation while another image is currently being edited.  This shows how useful the operation is.  For example, the user doesn't have to complete editing, and close the current image before they start working on another image.  Rather, separate images can be worked upon concurrently.

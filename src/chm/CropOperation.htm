Crop to Selection Operation

The Image Crop Operation (Image -->Crop to Selection) acts upon all layers in the image.  In order for a crop to be performed, the a selection must be defined by using either the Lasso Select Tool or the Rectangle Select Tool .  Once this area is defined, the Crop Operation will be available to you. If the Lasso Select Tool is used, the new image dimensions will be defined by the greatest X and Y values of the Lasso Selection .  If the Rectangle Select Tool is used, the new image dimensions will be the outline of the bounded area.

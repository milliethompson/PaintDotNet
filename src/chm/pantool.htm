Pan Tool

The Pan Tool allows the user to scroll around an image which is larger then the current viewable area.  For instance, if you have a 800x600 picture loaded into Paint.NET but only have a max resolution of 640x480, you can use the Pan Tool to view different areas of the image.  You may also use the .jpg"Tool  to view the complete image, as well.

Image Operations


The Image Operations alter the entire image which includes all the layers.  There are similar operations, the Layers Operations which only alter the specific active layer.  This critical difference should be noted, for if this detail is ignored, using these options will yield strange results from what was expected.  All Image Operations can be accessed through the Image drop-down menu.
 

The following are the Image Operations:

  • Flip
  • Crop
  • Resize
  • Canvas Size
  • Rotate
  • Zoom
  • Actual Size

Pencil Tool

The Pencil Tool (Tools -->Pencil) allows for simple drawing upon the active layer with a line of size 1.  A tool very similar to this is the Paint Brush which supports multiple sizes and fill patterns. 

 

 

 
The star on the side of his cheek was produced using the Pencil Tool.  Quick and dirty drawing is a very typical use of the Pencil tool. 
Important:  The Pencil Tool, along with all the other tools that alter the layer must be used within a selected area (if one so exists) created by either the Rectangle or the Lasso Selection Tools .  If you are attempting to use this tool outside of an existing selection, no alterations to the active layer will take place.

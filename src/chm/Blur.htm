Blur Effect

The Blur Effect (Effects-->Blur) uses the Gaussian blur algorithm to apply a convolution matrix to each pixel.  The size of this matrix is determined by the value selected by the user.  Each setting has an exponential effect on the layer (or region).  For instance, the bottom images are the result of applying different settings to the Blur Effect. 

 
Notice: The Blur Effect, along with the other effects will operate only in a selected region.  If no region is selected (as in the image below), the effect is applied to the entire layer.  Depending on the settings, the image size, and the internal clock speed of the CPU, these effects may take a considerable time to complete.

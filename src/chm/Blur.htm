Blur Effect

The Blur Effect (Effects-->Blur) uses the Gaussian blur algorithm to apply a convolution matrix to each pixel.  The size of this matrix is determined by the value selected by the user.  For instance, the bottom images are the result of applying different settings to the Blur Effect.

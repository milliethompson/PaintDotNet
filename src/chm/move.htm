Move Tool

The Move Tool (Tools-->Move) can move any number of selection regions around on the current layer.  Just have a selection region already active using the Rectangle or Lasso Selection Tool, and then click and drag.  That will move the selected region to any new location on the active layer.

This is the result of using the Move Tool on the selected region. 
Important:  In the above pictures, it appears that white space is left over where the selected region used to reside.  This is not true.  What is left behind is a transparent alpha color.  Therefore, what lies under the moved region is the layer currently under it (and the layer has to be active in order for you to see it).  Also, if the Move Tool is used without first selecting a region, the entire canvas (i.e. layer) will selected.

Undo Operation

The Undo Operation (Edit-->Undo) allows for undoing an action which could have been a mistake, or otherwise was determined to be unwanted.  The Undo Operation is also accessible from the History Window.  The images below show the gradual return to an original image by the continual usage of the Undo Operation.  Also, note that most actions you perform in Paint.NET can be undone.

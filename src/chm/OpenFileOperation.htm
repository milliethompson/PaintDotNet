Open Operation

The Open Operation (File-->Open) is very similar to the New Operation.  The main difference being is that rather then just clearing the canvas, you also select a new image to load.  If there is currently an active image, you will be prompted before it is cleared.
 
This is a sample open dialog box.  The file formats available to you are: Bitmap (*.bmp), Layered Bitmap (*.lbmp), Jpeg (*.jpg, *.jpeg, *.jpe and others),  PNG (*.png), TIFF (*.tiff, *.tif), and GIF (*.gif).

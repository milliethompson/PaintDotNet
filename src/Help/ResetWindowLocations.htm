Reset Window Locations

The Reset Window Locations (Window-->Reset Window Locations) feature resets the locations of all floating windows to their default workspace location.  Any floating window left in its default location will remain locked in place.  Any floating window moved from its default location will float independently above the application.
 

During the time one spends working on an image, the various windows may change locations many times.  In some instances, the user might actually end up misplacing a window. Notice in this example, the four windows have migrated from their initial default locations.

 After executing the Reset Window Locations option, the four windows return to their default locations; one at each of the four corner of the workspace.

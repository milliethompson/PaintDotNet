Edit Operations

The Edit Operations allow for easy manipulation of the image history, the selected region of the image, the selection itself, and the clipboard.
The following operations are found on the Edit Menu:
  • Undo
  • Redo
  • Cut
  • Copy
  • Paste
  • Paste in to New Layer
  • Erase Selection
  • Invert Selection
  • Select All
  • Deselect

Erase Selection Operation

The Erase Selection Operation (Edit-->Erase Selection) allows for the removal of a selected portion of the active layer. A portion of the layer must be selected using either the Rectangle Select or the Lasso Select.  Once the Erase Selection Operation is executed the selected region of the selected layer becomes 100% transparent.  In other words, the selected region is 'erased' from the layer.

The erase operation is similar to the cut operation, but differs in that the cut region is not saved to the clipboard. To better illustrate this operation, see the example below.

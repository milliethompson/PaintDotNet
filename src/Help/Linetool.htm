Line Tool

The Line Tool (Tools-->Line) operates very similar to the Pencil Tool.  The main difference being is that the Line Tool can use multiple widths and different drawing styles.  Also, unlike the Pencil Tool, the Line Tool doesn't draw a continuous path -- just a simple line from one point to another.

Pictured to the right is an example of multiple line widths and drawing patterns.  The line tool is very versatile.

 
Important:  The Line Tool, along with all the other tools that alter the layer must be used within a selected area (if one so exists) created by either the Rectangle or the Lasso Selection Tools.  If you are attempting to use this tool outside of an existing selection, no alterations to the active layer will take place.

In order to build properly, you must place dotnetfx.exe in this directory.
dotnetfx.exe is simply the .NET 1.1 Framework redistributable setup and is
available at:

http://www.microsoft.com/downloads/details.aspx?FamilyID=262d25e3-f589-4842-8157-034d1e7cf3a3&DisplayLang=en

Alternatively, assuming you are not interested in building the "full" setup
package, you may place any executable in here and name it "dotnetfx.exe."

Also, you must place the .NET Framework 1.1 SP1 files into the dotnet_1_1_sp1
directory. The files and download locations are:

* NDP1.1sp1-KB867460-X86.exe
  http://www.microsoft.com/downloads/details.aspx?familyid=A8F5654F-088E-40B2-BBDB-A83353618B38
  
* WindowsServer2003-KB867460-x86-ENU.EXE
  http://www.microsoft.com/downloads/details.aspx?familyid=AE7EDEF7-2CB7-4864-8623-A1038563DF23
